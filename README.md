# Date Utils

A minimal date calculator. Add or subtract days from a date, or count the days between two dates.

🌐 **[Live demo](https://gsandrini.github.io/date-utils)**

---

## Features

- **Add Days** - calculate a new date by adding or subtracting days
- **Count Days** - calculate years, months and days between two dates
- Native date picker - uses the browser/OS calendar
- Dark mode support
- Bilingual interface: 🇮🇹 Italian / 🇬🇧 English (auto-detected from browser language)
- Installable on desktop and mobile, works offline
- Zero runtime dependencies beyond Alpine.js (bundled locally)

---

## Usage

### As a web app

Open the [live demo](https://gsandrini.github.io/date-utils) in any modern browser  - no installation required.

### Install

The app is available as a **Progressive Web App (PWA)** and can be installed directly on your device.
 
1. Open the [live demo](https://gsandrini.github.io/date-utils) in your browser
2. Look for the **install icon** (⊕) in the address bar, or open the browser menu and select **"Install app"** / **"Add to Home Screen"**
3. Confirm the installation - the app will appear on your desktop or home screen and work just like a native app, even offline
